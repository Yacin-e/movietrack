# MovieTrack

MovieTrack is a personal movie watchlist and tracker built as a distributed microservices application. It consists of two Spring Boot 3 microservices (catalog-service and tracking-service) communicating via gRPC, a React 18 frontend, deployed on Minikube with Kubernetes and secured with Istio service mesh (mTLS + RBAC).
